# 데스티니 전대 — 게이트 인베이전

뱀파이어 서바이버즈 스타일의 **전대(戰隊) 디펜스** 게임. 단일 `index.html`로 동작하며 모바일/PC 브라우저 모두 지원합니다.

## 스토리

어느 날 현실에 **게이트**가 열리고, 이세계의 **마물**들이 쏟아져 나왔다. 같은 순간 세계 각지에서 미지의 힘에 눈뜬 **5인**이 모여 **전대**가 되어 맞선다.

## 게임 흐름

1. **캐릭터 선택** — 전대원 중 한 명을 고른다 (현재 2명 + 추가 예정 3명)
2. **이동만** 조작 — 화면 아무 곳이나 드래그 (PC: `WASD` / 방향키), **공격은 자동**
3. 마물을 처치해 보석을 모아 **레벨업** → 무기·능력 **3개 중 1개** 선택
4. **3분 생존 시 변신!** — 능력이 크게 강화된다
5. **5분 생존 시 게이트 봉쇄 성공**
