# 데스티니 판타지 — 마수의 밤

뱀파이어 서바이버즈 스타일의 **로그라이크 디펜스** 게임. 단일 `index.html` 파일로 동작하며 모바일/PC 브라우저 모두 지원합니다.

## 스토리

이세계로 떨어진 군인 **광민**. 무녀 **아카네**를 구하기 위해 끝없이 몰려오는 마수 무리를 **5분간** 버텨야 한다.

## 플레이 방법

- **이동:** 화면 아무 곳이나 드래그 (PC: `WASD` / 방향키)
- **공격:** 자동 — 광민이 가진 무기가 알아서 적을 타격
- 적을 처치해 떨어지는 보석을 모아 **경험치**를 얻고 **레벨업**
- 레벨업마다 무기·능력 **3개 중 1개** 선택
- **5분 생존 시 아카네 구출 성공!**

## 무기

| 무기 | 설명 |
|------|------|
| 🔫 소총 연사 | 가장 가까운 적에게 자동 사격 (탄환 수·관통 증가) |
| 🗡️ 대검 베기 | 전방을 베어냄 (Lv6부터 360° 회전 베기) |
| 💣 수류탄 | 가까운 적에게 던져 광역 폭발 |
| 🪬 아카네의 부적 | 동료 아카네의 부적이 주위를 회전하며 타격 |
| 🌀 수호 오라 | 몸 주위에 지속 피해 오라 |

## 능력 (패시브)

공격력 강화 · 신속(이동속도) · 광역 확장 · 속사(쿨다운) · 자력(수집 범위) · 강철 체력

## 실행

`index.html`을 브라우저에서 열면 바로 실행됩니다. 빌드/의존성 없음.
