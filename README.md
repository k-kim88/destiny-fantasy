# 데스티니 전대 — 게이트 인베이전

뱀파이어 서바이버즈 스타일의 **전대(戰隊) 디펜스** 게임. 단일 `index.html`로 동작하며 모바일/PC 브라우저 모두 지원합니다.

## 스토리

어느 날 현실에 **게이트**가 열리고, 이세계의 **마물**들이 쏟아져 나왔다. 같은 순간 세계 각지에서 미지의 힘에 눈뜬 **5인**이 모여 **전대**가 되어 맞선다.

## 게임 흐름

1. **캐릭터 선택** — 전대원 중 한 명을 고른다 (현재 2명 + 추가 예정 3명)
2. **전장 선택** — 게이트가 열린 장소를 고른다 (도심 폐허 / 마경의 숲 / 설원 전선 / 용암 지대). 배경은 16도트 픽셀 스타일로 코드 생성된다.
3. **이동만** 조작 — 화면 아무 곳이나 드래그 (PC: `WASD` / 방향키), **공격은 자동**
3. 마물을 처치해 보석을 모아 **레벨업** → 무기·능력 **3개 중 1개** 선택
4. **3분 생존 시 변신!** — 게임이 일시정지되며 변신 컷인(빛줄기·섬광·전대명) 후 능력이 크게 강화된다
5. **5분 생존 시 게이트 봉쇄 성공**

## 메타 성장 (강화 상점)

플레이로 **골드**를 모아(처치 수·생존 시간·변신·클리어 보너스) 타이틀의 **강화 상점**에서 영구 강화한다. 모든 캐릭터에 적용되며 브라우저에 저장(localStorage)된다.

- 최대 체력 · 공격력 · 이동속도 · 수집 범위 · 경험치 획득 · **부활**(쓰러져도 1회 부활)

## 보스 전리품 룰렛

보스를 처치하면 **전리품 상자**가 떨어진다. 획득하면 🎰 **룰렛**이 돌아가 무기·능력을 **1·3·5개** 랜덤 획득(뱀파이어 서바이버즈 방식).

## 능력 (패시브)

공격력 강화 · 신속 · 광역 확장 · 속사 · 자력 · 강철 체력

## 캐릭터 아트

`sprites/sheets/`의 4방향 스프라이트 시트(4열 애니 프레임 × 4행 정면/좌/우/후면)를 사용합니다. 이미지 로드 실패 시 코드로 그린 벡터 캐릭터로 자동 폴백합니다.

- `gwangmin_basic.png`, `kimchired.png`(광민 변신), `akane_basic.png`, `samonpink.png`(아카네 변신), `minmin_basic.png`, `goldbear.png`(민민 변신)
- 전장 배경은 코드 절차 생성(16도트 픽셀)이라 별도 이미지가 없습니다.

## 실행

`index.html`을 브라우저에서 열면 바로 실행됩니다. 빌드/의존성 없음.
