# 데스티니 크루세이더 (DESTINY CRUSADER)

RPG 게임. 빌드 없이 단일 `index.html`로 한 단계씩 만들어 갑니다.

빌드 도구 없이 **단일 `index.html`** 로 바로 도는 가장 단순한 구조에서 시작합니다.
한 번에 하나씩 기능을 추가하며 키워갑니다.

- **빌드 불필요**: `index.html` 하나가 전부 (HTML+CSS+JS)
- **외부 의존성 없음**: 시스템 폰트 사용, 오프라인 동작
- **배포**: `main`에 푸시하면 GitHub Actions가 `https://k-kim88.github.io/destiny-fantasy/` 에 자동 배포

## 로컬에서 보기

`index.html` 파일을 브라우저로 열기만 하면 됩니다. (또는 `python3 -m http.server` 후 접속)
