# 광민(Gwangmin) 전투 스프라이트

이 폴더에 아래 **16개 PNG**(배경 투명, 한 캐릭터만)을 넣으면 전투에서 자동 적용됩니다.
권장: 정사각형 캔버스(예: 256×256), 모두 같은 크기·같은 방향(정면 또는 오른쪽). 파일명이 정확해야 합니다.

| 게임 상황 | 파일명 | 컨셉 시트 번호 |
| --- | --- | --- |
| 평상시(필드) | `idle.png` | 1. IDLE NEUTRAL |
| 전투 대기 | `stance.png` | 2. IDLE BATTLE STANCE |
| 걷기 1 | `walk1.png` | 3. WALK 1 |
| 걷기 2 | `walk2.png` | 4. WALK 2 |
| 달리기 | `run.png` | 5. RUN |
| 일반공격(시작) | `slash_start.png` | 6. ATTACK SLASH START |
| 일반공격(타격) | `slash_impact.png` | 7. ATTACK SLASH IMPACT |
| 스킬(강타) | `heavy.png` | 8. HEAVY ATTACK / SKILL |
| 방어 | `guard.png` | 9. GUARD / DEFEND |
| 피격 | `hit.png` | 10. HIT REACTION |
| 큰 피격 | `staggered.png` | 11. STAGGERED / HURT |
| 기절 | `stunned.png` | 12. STUNNED / DIZZY |
| 다운 | `knockdown.png` | 13. KNOCKED DOWN |
| 승리 | `victory.png` | 14. VICTORY POSE |
| 마법/스킬 시전 | `casting.png` | 15. CASTING POSE |
| 사망 | `death.png` | 16. DEATH / FAINTED |

> 파일을 넣고 main에 올리면(또는 알려주시면) 그대로 표시됩니다. 파일이 없으면 임시 이모지(🤺)로 표시됩니다.
